# Password-Validator-and-Generator
Users can use this online platform to generate strong passwords or even create one for themselves by checking few conditions for a strong password. 
Password Validator & Generator
This project is a web-based tool designed to enhance password security by providing users with two key functionalities: validation and generation.

Features:
Password Validation:

Users can enter their passwords to verify compliance with essential security criteria.
The tool checks for:
Minimum length (at least 8 characters)
Inclusion of digits
Use of special characters (e.g., !@#$%^&*)
Presence of both uppercase and lowercase letters
Real-time feedback is provided, allowing users to adjust their passwords accordingly.
Password Generation:

Users can generate strong, random passwords based on customizable settings.
Options include specifying password length and choosing character types such as:
Uppercase letters
Lowercase letters
Numbers
Special characters
This feature enables users to create secure passwords effortlessly.
User-Friendly Interface:

The application features an intuitive layout suitable for both beginners and advanced users.
A toggle button allows users to show or hide their typed passwords, enhancing usability during validation.
This tool aims to promote good password practices and improve overall online security for users by providing easy access to validation and generation functionalities.

Getting Started:
Clone the repository and open index.html in your browser to start using the Password Validator & Generator.
